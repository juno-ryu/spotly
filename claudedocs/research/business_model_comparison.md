## 2. 오픈업 비즈니스 모델 해부

### "무료"의 진짜 이유 — 대출 중개 리드 제너레이션

오픈업이 무료인 건 자선사업이 아니다. 핀다의 **핵심 비즈니스는 대출 중개**이고, 오픈업은 그 파이프라인의 입구다.

```
┌─────────────────────────────────────────────────────────────┐
│  핀다-오픈업 수익 흐름도                                       │
│                                                             │
│  [무료 상권분석]                                              │
│       ↓ 사용자 유입                                          │
│  [창업 의지 확인 + 위치 탐색]                                  │
│       ↓ 구매력 있는 예비 창업자 식별                            │
│  [창업 자금 필요 시점]                                        │
│       ↓ 자연스럽게 핀다 대출 연계                              │
│  [핀다 대출 중개] → 88개 금융사 비교                           │
│       ↓                                                     │
│  [금융사 → 핀다에 중개 수수료 지급]  ← 💰 여기서 돈을 번다     │
└─────────────────────────────────────────────────────────────┘
```

**증거**: 핀다의 개인사업자 대출 신청 비중이 2023년 15.1% → 2024년 48%로 **3배 이상 급증**. 개인사업자 약정 금액 1조원 돌파. 오픈업 인수 이후의 직접적 성과.

### 수익원 구조

| 수익원 | 비중(추정) | 메커니즘 |
|--------|----------|---------|
| 대출 중개 수수료 | 70%+ | 오픈업 → 핀다 대출 연계 → 금융사 수수료 |
| 오픈업 프로 (B2B) | 20% | 프랜차이즈 본사 구독 (월 99만원~) |
| 데이터/광고 | 10% | 상권 데이터 판매, 제휴 광고 |

### 데이터 경쟁력의 원천

오픈업이 압도적인 이유는 **카드사+통신사 데이터 결합**에 있다:
- 카드사 데이터: 실제 결제 기반 매출 추정 (현금 제외)
- 통신사 데이터: 유동인구 실시간 패턴
- 이 두 데이터는 **개인이나 소규모 스타트업이 절대 확보할 수 없음** (대기업 간 B2B 계약 필요)

### 오픈업의 약점

1. **카드 매출 = 추정치**. 현금 거래, 지역화폐, 건보 급여 미반영. 사용자의 15%가 자기 정보 비공개 원함
2. **비공개 절차가 불편**. 사업자등록증 제출 + 90일마다 갱신 + 비공개 해제 후 30일 재신청 불가
3. **소상공인 반발**. "나도 모르는 사이에 내 매출이 공개됐다" → 개인정보 논란 지속
4. **무료 = 사용자가 상품**. 사용자의 창업 의도/위치/업종 데이터가 대출 영업에 활용됨
5. **기본 무료 분석의 깊이가 제한적**. 진짜 심화 분석은 오픈업 프로(B2B, 월 99만원~)

## 3. 마이프차 비즈니스 모델 해부

### 프랜차이즈 특화 양면 플랫폼

마이프차는 상권분석 서비스가 아니라 **프랜차이즈 창업 플랫폼**에 가깝다. 상권분석은 여러 기능 중 하나.

```
┌───────────────────────────────────────────────────────┐
│  마이프차 양면 플랫폼 구조                               │
│                                                       │
│  [예비 창업자]          ←→          [프랜차이즈 본사]    │
│   · 브랜드 비교                      · 가맹점 모집 광고  │
│   · 상권분석 (마이프차 지도)           · 창업 문의 리드   │
│   · 가게 인수 거래                    · 가맹점 관리 도구  │
│   · 창업 상담 연결                    · 매출 분석 대시보드 │
│                                                       │
│              💰 수익 발생 지점                          │
│   · B2C: 마이프차 지도 구독 (월 59,400원)               │
│   · B2B: 프랜차이즈 본사 광고비/노출비                   │
│   · 거래: 가게 인수 중개 (수수료 미공개)                  │
└───────────────────────────────────────────────────────┘
```

### 수익원 구조

| 수익원 | 비중(추정) | 메커니즘 |
|--------|----------|---------|
| 프랜차이즈 본사 광고 (B2B) | 50%+ | 플랫폼 내 브랜드 노출, 상위 배치, 창업 문의 리드 판매 |
| 마이프차 지도 구독 (B2C) | 30% | 월 59,400원 개인 구독 |
| 가게 인수 거래 중개 | 10% | 운영 중인 프랜차이즈 거래 플랫폼 |
| 기타 | 10% | B2B 상권분석 솔루션 제공 |

### 마이프차의 핵심 자산

**공정거래위원회 정보공개서 1만개+ 통합**. 프랜차이즈별 가맹비, 평균 매출, 폐점률, 가맹점 증감 추이 등을 비교할 수 있는 곳은 마이프차가 유일에 가까움. 이 데이터 자체가 진입장벽.

### 마이프차의 약점

1. **일회성 사용 문제**. 창업 완료하면 재방문 이유 없음 → LTV 한계
2. **프랜차이즈 특화 = 시장 한정**. 개인 독립 창업자는 타겟 밖
3. **상권분석은 보조 기능**. 핵심은 브랜드 비교/창업 상담이지 상권분석이 아님
4. **월 59,400원 구독은 높은 진입장벽**. 예비 창업자가 아직 수익도 없는데 월 6만원 구독은 심리적 부담
5. **AI 분석 없음**. 데이터를 보여주기만 할 뿐 "여기서 해도 되나요?"에 답 못함

## 4. 우리 비즈니스 모델의 장단점

### 우리 모델 구조

```
┌─────────────────────────────────────────────────┐
│  우리 서비스 수익 모델                             │
│                                                 │
│  [무료 기본 분석]                                 │
│   · 5대 지표 점수 (블러 처리)                     │
│   · 주변 업체 목록 (기본)                         │
│       ↓ 호기심 유발                              │
│  [유료 상세 분석] ← 💰 직접 과금                  │
│   · 건당 ₩3,900                                 │
│   · 월 ₩9,900 (10건)                            │
│   · 월 ₩19,900 (무제한)                          │
│       ↓ 제공                                    │
│  [AI 리포트] Claude가 생성                       │
│   · verdict (추천/조건부/주의/비추천)              │
│   · strengths + risks + recommendations         │
│   · PDF 다운로드                                │
│                                                 │
│  [Phase 3 확장]                                  │
│   · Consultant ₩49,900/월 (AI 대화형)            │
│   · Enterprise B2B (별도 협의)                   │
└─────────────────────────────────────────────────┘
```

### 4.1 장점 — 오픈업/마이프차 대비

#### ✅ 1. 수익 모델의 투명성과 건전성

| 우리 | 오픈업 | 마이프차 |
|------|--------|---------|
| 사용자가 직접 분석 비용을 지불 | 사용자가 "상품" (대출 리드) | 프랜차이즈 본사가 "고객" |
| 이해관계 충돌 없음 | **이해관계 충돌 가능** — 대출이 필요한 쪽으로 유도할 유인 | 광고주 브랜드를 우선할 유인 |

오픈업은 무료지만, 사용자의 창업 데이터가 대출 영업에 활용된다. 우리는 **"돈을 내는 만큼 객관적인 분석을 받는다"**는 명확한 가치 교환이 가능.

이걸 마케팅 메시지로 쓸 수 있음: *"우리는 당신에게 대출을 팔지 않습니다. 분석을 팔 뿐입니다."*

#### ✅ 2. 가격 포지셔닝의 절묘함

```
무료 ──── ₩3,900 ──────────────────── ₩59,400 ──── ₩99만
(오픈업)   (우리)                      (마이프차)     (오픈업 프로)
           ↑
     "커피 한 잔 가격에 AI가 해석"
```

- 무료 서비스 사용자 중 "좀 더 깊은 분석"이 필요한 사람이 ₩3,900을 내는 건 **심리적 장벽이 거의 없음**
- 마이프차 월 ₩59,400은 아직 수익이 없는 예비 창업자에겐 부담
- 나이스비즈맵 ₩55,000/건은 진지한 사업자용 — 그 사이가 비어있음
- **건당 원가 ₩500 이하 → 마진율 ~85%**로 수익성 우수

#### ✅ 3. AI 리포트의 차별성

| 기능 | 오픈업 | 마이프차 | 우리 |
|------|--------|---------|------|
| AI 분석 | "AI 10초 리포트" (기본) | ❌ 없음 | **Claude AI 맞춤 리포트** |
| verdict | ❌ | ❌ | ✅ 추천/조건부/주의/비추천 |
| 맞춤 조언 | 제한적 | ❌ | ✅ strengths + risks + recommendations |
| PDF 리포트 | ❌ (프로만) | ❌ | ✅ (Pro 구독) |

오픈업의 "AI 10초 리포트"는 정형화된 요약 수준. 우리는 Claude가 **분석 맥락을 이해하고 판단까지 내려주는 진짜 AI 리포트**를 제공. "이 자리에서 치킨집을 하면 안 되는 3가지 이유"처럼 구체적.

#### ✅ 4. 스코어링 v2의 "신뢰도 점수" (경쟁사에 없는 기능)

오픈업도 마이프차도 **데이터 품질에 대한 투명성이 없음**. 카드 매출은 현금 거래를 빼먹고, 추정 매출의 오차 범위도 안 알려줌.

우리는 스코어링 v2에서 **신뢰도(Confidence Score)**를 공개:
- "72점 🟢높음" → 데이터 충분, 신뢰 가능
- "68점 🔴낮음" → 이 지역은 데이터가 부족하니 추가 조사 권장

**"솔직함이 차별화"** — 경쟁사가 "정확합니다"라고 주장할 때, 우리는 "이 점수의 신뢰도는 0.62입니다"라고 말하면 오히려 전문성과 신뢰를 얻을 수 있음.

#### ✅ 5. 기술 스택 유연성

| 항목 | 오픈업 | 마이프차 | 우리 |
|------|--------|---------|------|
| AI 모델 | 자체/미공개 | 없음 | **Anthropic Claude** (최신 모델 즉시 적용) |
| 모델 업그레이드 | 자체 개발 필요 | N/A | Haiku→Sonnet→Opus 즉시 전환 |
| Phase 3 확장 | 핀다 생태계에 종속 | 프랜차이즈에 종속 | **AI 컨설턴트 모드** (MCP 기반 대화형) |

Claude API 기반이므로 Anthropic이 모델을 개선할 때마다 **개발 없이 분석 품질이 자동 향상**됨. 오픈업은 자체 AI를 유지보수해야 함.

#### ✅ 6. 모바일 UX 차별화

5단계 대화형 온보딩 (Step 1~5) + 지도 중심 인터랙션은 **오픈업/마이프차 둘 다 데스크톱 대시보드 기반**인 것과 대조적. 프리페치 전략 적용 시 Step 5 결과까지 0.1~1.5초 (경쟁사 대비 5~10배 빠름).

### 4.2 약점 — 오픈업/마이프차 대비

#### ❌ 1. 데이터 소스의 근본적 열세 (가장 치명적)

| 데이터 | 오픈업 | 마이프차 | 우리 |
|--------|--------|---------|------|
| 카드 매출 | ✅ (1.5억건) | ✅ (카드 매출 기반) | ❌ **없음** |
| 통신 유동인구 | ✅ (통신사 결합) | ❌ | ❌ **없음** |
| 공공 API | ✅ | ✅ | ✅ |
| 프랜차이즈 정보공개서 | ✅ | ✅ (1만개+) | ❌ **없음** |
| NPS 국민연금 | 일부 | 일부 | ✅ |
| 부동산 실거래 | ✅ | ✅ | ✅ |
| KOSIS 인구 | ✅ | ✅ | ✅ |

**카드 매출 데이터 없이 "이 자리의 예상 매출"을 추정할 수 없다.** 이것이 가장 큰 약점. 오픈업이 사용자 만족도 1위 기능으로 "관심 사업장의 상세 매출 데이터(60.3%)"를 뽑은 이유. 우리는 이 기능을 **제공할 수 없다**.

> **완화 전략**: 카드 매출 대신 **NPS 고지금액(crrmmNtcAmt) 기반 급여 추정 → 지역 소득수준 → 소비력 간접 추정** 경로로 대체. 정확도는 떨어지지만, "추정 매출"이 아닌 **"소비력 지수"**라는 다른 프레임으로 제시하면 경쟁 회피 가능.

#### ❌ 2. "무료" 경쟁사와의 본질적 싸움

오픈업이 무료인 건 대출 수수료로 벌기 때문. **우리는 대출 중개가 없으므로 무료로 제공할 이유가 없고, 무료로 제공하면 수익 모델이 없다.**

사용자 입장에서 "오픈업은 무료인데 왜 여기는 3,900원이지?" 라는 질문이 반드시 나옴.

> **완화 전략**: "무료 = 당신이 상품"이라는 인식을 심어야 함. 오픈업의 개인정보 논란(매출 공개 반발, 비공개 절차 불편)을 우리의 마케팅 포인트로 활용:
> - *"오픈업은 당신의 매출을 공개합니다. 우리는 당신의 미래를 분석합니다."*
> - *"3,900원은 분석 비용입니다. 무료 서비스의 대가는 당신의 데이터입니다."*

#### ❌ 3. 브랜드 인지도와 신뢰도 제로

| 신뢰 요소 | 오픈업 | 마이프차 | 우리 |
|-----------|--------|---------|------|
| 모기업 | 핀다 (644억 투자) | 130억 투자 | 없음 |
| 언론 보도 | 다수 | 다수 | 없음 |
| 사용자 수 | 1200+ 설문 응답 | MAU 30만 | 0 |
| 만족도 | 94% | 미공개 | 미측정 |

예비 창업자가 수천만원을 투자하는 의사결정에 "처음 보는 서비스"의 3,900원 분석을 믿을 수 있을까?

> **완화 전략**:
> - 무료 기본 분석으로 먼저 신뢰 구축 → 유료 전환 유도
> - 분석 결과에 "데이터 출처: 국민연금공단, 국세청, 한국부동산원, 통계청" 명시하여 공공데이터 기반 신뢰성 강조
> - 신뢰도 점수 공개 자체가 전문성의 증거

#### ❌ 4. 네트워크 효과 부재

오픈업: 사용자가 많을수록 → 대출 중개 데이터 축적 → 더 정교한 금융 상품 추천 → 더 많은 사용자
마이프차: 창업자가 많을수록 → 프랜차이즈 본사 광고 유치 → 더 많은 브랜드 정보 → 더 많은 창업자

**우리**: 사용자가 많아져도 비즈니스 모델이 선순환하는 구조가 약함. 건당 과금은 **거래형(transactional)**이라 네트워크 효과가 없음.

> **완화 전략**: Phase 3의 AI 컨설턴트 모드 + 분석 이력 기반 "이 지역 트렌드" 기능을 통해 축적 데이터의 가치를 높일 수 있음. 장기적으로 분석 결과 자체가 독자적 데이터셋이 됨.

#### ❌ 5. 프랜차이즈 정보의 부재

마이프차는 공정위 정보공개서 1만개를 통합하여 **가맹비, 로열티, 평균 매출, 폐점률, 가맹점 증감** 등을 비교할 수 있음. 이건 프랜차이즈 창업자에게 결정적인 데이터인데, 우리는 **이 데이터가 전혀 없음**.

> **완화 전략**: 공정위 정보공개서 데이터는 공공데이터로 확보 가능. Phase 2.5~3에서 프랜차이즈 정보 연동을 검토. 단, 1만개 브랜드 통합은 상당한 크롤링/정제 작업이 필요.

## 6. 전략적 시사점

### 6.1 우리가 절대 따라갈 수 없는 것 (수용)

1. **카드사 매출 데이터** — B2B 계약 필요, 스타트업 단계에서 불가능
2. **통신사 유동인구 데이터** — 동일
3. **대출 중개 수수료로 무료 제공** — 금융업 라이센스 + 제휴 필요

→ 이 3가지에서 경쟁하지 말 것. 다른 축으로 싸워야 함.

### 6.2 우리가 즉시 공략할 수 있는 약점

| 경쟁사 약점 | 우리의 기회 |
|------------|-----------|
| 오픈업의 개인정보 논란 | **"우리는 당신의 데이터를 팔지 않습니다"** 마케팅 |
| 마이프차의 높은 구독료 (₩59,400) | **₩3,900으로 동급 분석** 가격 파괴 |
| 둘 다 AI verdict 없음 | **"추천/비추천" 판단까지 해주는 건 우리뿐** |
| 둘 다 데이터 품질 불투명 | **신뢰도 점수 공개** → 전문성 어필 |
| 둘 다 모바일 UX 미흡 | **대화형 온보딩 + 1.5초 결과** |
| 오픈업 무료는 "기본"만 | **유료지만 "프리미엄" AI 리포트 포함** |

### 6.3 포지셔닝 선언

```
오픈업 = "무료로 데이터를 보여주는 대출 플랫폼"
마이프차 = "프랜차이즈 브랜드를 비교하는 플랫폼"

우리 = "₩3,900에 AI가 '여기서 해도 되나요?'에 답하는 서비스"
```

**경쟁 축을 "데이터 양"에서 "해석의 질"로 전환.** 데이터 양에서는 이길 수 없지만, "그래서 어떻게 해야 하는데?"에 대한 답변 품질에서는 이길 수 있음.

## 7. 리스크 시나리오

### 시나리오 A: 오픈업이 AI 리포트를 무료로 확장 (확률: 중)

핀다가 Claude/GPT 기반 AI 리포트를 오픈업 기본에 무료로 추가하면?

**대응**:
- 우리의 AI 리포트는 **5대 지표 100점 만점 + 신뢰도 + 맞춤 verdict**라는 구조화된 분석. 오픈업이 "무료 AI 요약"을 추가해도 깊이 차이가 남
- Phase 3 AI 컨설턴트 모드(대화형)로 추가 차별화

### 시나리오 B: 소상공인365가 AI 기능 추가 (확률: 낮)

정부 서비스에 AI가 추가되면?

**대응**:
- 공공 서비스는 업데이트가 느리고, 정부 AI는 보수적 표현만 사용
- "비추천"이라는 판단을 정부 서비스가 내리기는 정치적으로 어려움
- 우리는 **솔직한 verdict**가 가능

### 시나리오 C: 마이프차가 가격을 대폭 인하 (확률: 중)

마이프차가 ₩59,400 → ₩9,900으로 인하하면?

**대응**:
- 마이프차는 프랜차이즈 특화 → 독립 창업자 시장에서는 경쟁 안 됨
- AI 리포트가 없으므로 해석 차별화 유지
- 우리는 건당 ₩3,900 → 한 번 써보기 쉬움 (구독 부담 없음)
