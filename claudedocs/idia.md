## 7. 🔍 기업 실사(Due Diligence) AI 플랫폼

**핵심 아이디어**: 사업자번호 하나만 입력하면 "이 회사 거래해도 될까?"를 3분 안에 판단

**활용 API/MCP 조합**:
| 도구 | 역할 |
|------|------|
| 🔌 NTS Business Verification MCP | 사업자 진위확인 + 영업/휴업/폐업 상태 |
| 🔌 FSC Financial Info MCP | 매출·영업이익·부채비율 재무분석 |
| 🔌 NPS Business Enrollment MCP | 직원 수 변동 추이, 추정 평균 급여 |
| 한국은행 경제통계 API | 업종별 평균 지표와 비교 분석 |
| 네이버 CLOVA Studio | AI 리포트 자동 생성 |

**수익 모델**:

- **건당 과금** (3,000~10,000원): 1회 기업 실사 리포트
- **월정액 B2B** (월 99,000~499,000원): 무제한 조회
- **VC/PE 펀드 대상 엔터프라이즈**: 투자 심사 자동화 (연 1,000만원+)
- **프랜차이즈 가맹 심사**: 가맹본부 대상 SaaS

**타겟 고객**: VC/PE 투자심사역, 스타트업 CFO, 프랜차이즈 본사, B2B 영업팀, 법무법인

**왜 유망한가**: 현재 기업 실사는 수작업으로 3~7일 걸리는데, MCP 3개 조합으로 즉시 자동화 가능. 크레딧잡(CreditJob) 같은 기존 서비스가 월 수억 매출을 내고 있어 시장이 검증되었습니다.

**핵심 아이디어**: 나라장터에서 "우리 회사에 맞는 입찰"을 AI가 자동으로 찾아주고 제안서 초안까지

| 🔌 FSC Financial Info MCP | 경쟁사 재무 비교 분석 |
| 네이버 CLOVA Studio | 제안서 초안 AI 작성 |
| 통계청 KOSIS | 산업별 평균 낙찰률 분석 |

**수익 모델**:

- **기본 구독** (월 49,000원): 맞춤 입찰 알림 + 경쟁 분석

**타겟 고객**: IT 중소기업, 건설사, 용역업체 (정부 조달 시장 약 150조원)

**왜 유망한가**: 정부 조달 시장은 연 150조원 규모인데, 중소기업의 80%가 나라장터 공고를 수작업으로 모니터링하고 있습니다. PPS MCP로 이 과정을 완전 자동화할 수 있습니다.

**수익 모델**:

## 10. ⚗️ 화학물질 안전관리 SaaS (제조업 특화)

**핵심 아이디어**: 공장에서 쓰는 화학물질의 MSDS를 AI로 자동 관리하고 규제 준수를 보장

**활용 API/MCP 조합**:
| 도구 | 역할 |
|------|------|
| 🔌 MSDS Chemical Info MCP | 화학물질 16개 섹션 전체 정보 자동 조회 |
| Air Korea 대기오염정보 API | 작업장 주변 환경 모니터링 |
| 기상청 날씨 API | 기온·습도에 따른 화학물질 위험도 변동 |
| 네이버 CLOVA Studio | 안전교육 자료 AI 생성 |

**수익 모델**:

- **SaaS 구독** (월 99,000~499,000원/공장): MSDS 자동 관리 + 규제 알림
- **안전교육 콘텐츠** (건당 50,000원): AI 기반 맞춤 안전교육 자료 생성
- **컨설팅**: 화학물질 규제 대응 자문 (시간당 200,000원)
- **보험사 제휴**: 안전 점수 기반 산재보험료 할인

**타겟 고객**: 제조업 공장 (국내 약 7만개소), 화학회사, 반도체·배터리 공장

**왜 유망한가**: 화학물질관리법(화관법) 강화로 MSDS 관리 의무가 확대되는데, 중소 제조업체는 전담 인력이 없어 자동화 수요가 큽니다. MSDS MCP가 16개 섹션을 모두 커버하므로 서비스 구축이 즉시 가능합니다.

# TOP 3 추천

### 🥇 1순위: 기업 실사 AI (7번)

→ MCP 3개만으로 즉시 구현 가능, 초기 비용 최저, B2B 단가 높음. 사업자번호 하나로 3분 안에 기업 분석 리포트 생성.

### 🥈 2순위: 소상공인 창업 분석기 (11번)

→ 매년 100만 명 창업, TAM 거대. NPS로 "같은 동네 같은 업종 사업장이 직원을 늘리고 있는지"를 추적하면 기존 상권분석 서비스를 압도.

### 🥉 3순위: B2B 거래처 신용 모니터링 (15번)

→ API 4개로 구현 가능, 월 반복 매출(MRR) 구조. "직원 수 3개월 연속 감소" 감지 = 공식 폐업 공시보다 선행하는 강력한 위험 지표.

> **공통 핵심**: MCP 서버의 가장 강력한 조합은 **NTS(진위확인) + NPS(직원/급여) + FSC(재무)**입니다. 이 3개를 기반으로 하면 기업 분석 관련 거의 모든 서비스를 만들 수 있습니다.
