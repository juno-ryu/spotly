## 📋 무료 버스 API 탐색 완료 보고

조건을 만족하는 **3개의 주요 무료 버스 API**를 발굴했습니다.

---

## 🥇 **1순위: 서울시 버스 정류소 정보 조회 API (OA-1094)**

### API 정보
- **제공 기관**: 서울특별시 교통정보시스템  
- **API 유형**: OpenAPI (7개 서비스)  
- **무료 여부**: ✅ 완전 무료  
- **인증 방식**: 공유자원포털에서 API 키 발급 (무료)

### 핵심 메서드
1. **정류소명으로 검색** (Stop name search)
2. **정류소 ID로 조회** (Stop ID inquiry)  
3. **👉 정류소별 경유노선 목록 조회** ← **필수 기능 완벽 만족**
4. **좌표 기반 근처 정류소 검색** ← **위경도 기반 검색 가능**
5. 첫/막차 시간 조회
6. 접근성 정류소 조회

### 제공 필드
- 정류소명, 정류소ID(ARS-ID), 경유노선명, 노선 수
- 위도/경도 (WGS84 좌표계, EPSG-5179)
- 정류소 유형 (일반/저상/야간 등)

### 갱신 주기
비정기 (자료 변경시 수시 반영)

### 호출 방식 (예시)
```
정류소별 경유노선: 
http://ws.bus.go.kr/api/rest/stationinfo/getRouteByStation?arsId=xxxxx

→ 응답: 해당 정류소에 정차하는 모든 노선 목록
```

### 주의사항
- ⚠️ **ARS-ID 5자리**: CSV 파일의 4자리 코드는 앞에 0 추가 필요 (1234 → 01234)
- 위경도 기반 검색은 **OA-15067**(위치정보 데이터셋) 또는 직접 API 조회로 정류소 ID 획득 필요

## 🥈 **2순위: 국토교통부 TAGO 버스정류소정보 API (data.go.kr)**

### API 정보
- **제공 기관**: 국토교통부 모빌리티총괄과  
- **서비스명**: BusSttnInfoInqireService  
- **무료 여부**: ✅ 완전 무료  
- **API 키**: 공공데이터포털에서 발급

### 핵심 메서드
1. **좌표 기반 근접 정류소 목록** (getCrdntPrxmtSttnList)
   - 파라미터: `gpsLati`, `gpsLong` (WGS84)
   - 반경: 500m 이내
   - ✅ 위경도 기반 검색 **완벽 지원**

2. **정류소별 경유 노선** (getSttnThrghRouteList)
   - 파라미터: `nodeId` (정류소 ID)
   - ✅ 정류소 경유 노선 **완벽 지원**

3. 정류소번호 검색
4. 도시코드 목록 조회

### 제공 필드
- 정류소명, 정류소 고유ID(nodeId), 경유노선 상세
- 위도/경도 (WGS84)
- 도시 코드

### 트래픽 제한
- 개발계정: 10,000건/일
- 운영계정: 활용사례 등록 후 증가 가능
- **자동 승인** (신청 후 즉시 사용 가능)

### 호출 방식 (예시)
```
기본 엔드포인트:
http://apis.data.go.kr/1613000/BusSttnInfoInqireService/getCrdntPrxmtSttnList

파라미터:
?serviceKey=YOUR_KEY&gpsLati=37.566&gpsLong=126.978&cityCode=26&numOfRows=10
```

### 주의사항
- TAGO는 각 지역 BIS(버스정보시스템)에 연결되어 있음
- 도시마다 데이터 품질/갱신 주기가 다를 수 있음

## 🥉 **3순위: 경기도 버스 Open API (경기데이터드림)**

### API 정보
- **제공 기관**: 경기도청  
- **제공 포탈**: 경기데이터드림 (data.gg.go.kr)  
- **무료 여부**: ✅ 완전 무료

### 핵심 메서드
1. **정류소 검색** (Station Search)
   - 정류소명/번호 검색
   - **반경 200m 이내 근처 정류소 검색 가능**

2. **정류소별 경유노선 조회**
   - 특정 정류소를 지나는 모든 노선 목록

3. 노선 상세 조회
4. 실시간 버스 위치/도착정보

### 제공 필드
약 11개 정보항목 (정류소명, 위경도, 경유노선 등)

### 데이터 규모
약 110만 건의 경기도 정류소 데이터

### 주의사항
- ⚠️ 경기도 지역 전용 (창업 입지가 경기도인 경우 적극 활용)
- API 명세서 다운로드 및 테스트 필요
