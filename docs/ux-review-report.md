# 창업 분석기 UX 리뷰 리포트

**탐색 일시**: 2026-02-28
**탐색 경로**: 홈 → 업종 선택(카페) → 지역 선택(홍대) → 지도 핀 → 분석 결과 → 결제 모달 → AI 리포트

---

## ✅ 잘한 포인트

### 1. 온보딩 플로우가 자연스럽다
"창업을 준비중이신가요?" → "네, 알아보고 있어요 👋" 형식의 대화형 진입이 딱딱하지 않고 친근하다. 서비스 색깔이 잘 드러남.

### 2. 업종/지역 선택에 빠른 선택 칩 제공
"핫한 창업 아이템", "지금 핫한 창업지역" 같은 추천 칩이 있어서 뭘 선택해야 할지 모를 때 즉시 진입 가능. 검색 + 추천의 병용 구조가 좋음.

### 3. 지도에서 반경이 시각적으로 표시됨
300m 반경 원형 + 경쟁업체 핀이 지도에 바로 보여서 "이 동네 카페가 이렇게 많구나"를 직관적으로 인지 가능. 데이터를 눈으로 먼저 느끼게 하는 구조.

### 4. 분석 결과 카드가 읽기 좋다
경쟁강도 F / 상권 활력도 A 같이 등급 라벨 + 한줄 요약 + 근거 데이터 3단 구조가 일관되고 스캔하기 편함.

### 5. 결제 모달의 무료 vs 유료 비교표
체크박스로 무료/AI 리포트 기능 차이를 한눈에 보여주는 구조. 유료 전환 설득력이 있음.

### 6. AI 리포트 내용 자체가 훌륭하다
- "종합 판단 / 강점 / 위험 요소 / 제언 / 상세 분석" 5단 구조가 명확함
- "42m마다 카페 1개", "폐업률 6% vs 개업률 3%" 같은 구체적 수치로 설득력 있음
- 제언이 실용적임 ("공실 확인 전에 인근 카페 3~5곳 실제 방문해서...")
- 마지막 면책 문구("공공 데이터 기반 참고 자료")로 법적 리스크도 커버

## ❌ 개선 포인트

### 1. 홈 화면 여백이 너무 많다
텍스트 2줄 + 버튼 1개가 1400px 높이 화면에 덩그러니 있음. 모바일이면 괜찮지만 데스크탑에서 허전함. 서비스 소개, 신뢰 지표(분석 건수, 리뷰 등) 넣을 여지가 충분함.

### 2. 분석 결과 페이지에서 지도가 절반 이상 차지한다
스크롤 해야 결과 카드가 보이는데, 지도가 화면 상단 절반을 고정으로 잡고 있어서 핵심 정보(점수, 인사이트)까지 스크롤이 필요함. 지도를 접을 수 있게 하거나 높이를 줄이는 게 나을 것 같음.

### 3. "AI 맞춤 리포트 잠금 해제" CTA가 너무 아래 숨어있다
CTA가 페이지 최하단 고정 버튼인데, 분석 결과 카드를 보면서 "이거 더 보고 싶다"는 욕구가 생기는 시점에 버튼이 항상 보여야 전환율이 높아짐. 지금은 스크롤 내리기 전까지 CTA 존재 자체를 모를 수 있음.

### 4. 로딩 상태 피드백이 없다
"분석 시작하기" 클릭 후 결과 페이지로 바로 넘어가는데, 그 사이 로딩 인디케이터나 "분석 중..." 화면이 없음. 데이터 수집이 오래 걸릴 경우 빈 화면이 뜨거나 사용자가 클릭을 반복할 수 있음.

### 5. 결제 모달 후기가 너무 짧고 신뢰도가 낮다
한 줄 후기 + 1,247건 표시가 있는데, 후기 내용이 너무 짧고 이름/직업만 표시돼서 신뢰감이 약함. 사진이나 별점, 구체적 수치("이 리포트 보고 계약 포기, 월세 500만원 아꼈어요") 같은 후기가 전환율 높임.

### 6. 경쟁강도 F인데 분석 계속 진행되는 흐름
경쟁강도 F (매우 위험) 상권에서도 결제 유도로 바로 이어짐. "이 상권은 위험합니다, 대신 이 지역은 어떨까요?" 같은 대안 제안 플로우가 있으면 서비스 신뢰도가 올라가고 재사용도 유도됨.

### 7. AI 리포트 페이지 하단 여백이 너무 크다
리포트 본문 아래로 빈 공간이 엄청나게 남아있음. PDF 다운로드 버튼이 페이지 최하단에 외롭게 있음. 공유 버튼, 다른 지역 분석하기 CTA를 이 공간에 추가하면 좋을 것 같음.

### 8. AI 리포트에 시각 요소가 없다
텍스트만 있음. 경쟁강도 F / 상권활력도 A 등 수치를 차트나 게이지로 보여주면 가독성과 공유 욕구 모두 높아짐. 분석 결과 페이지에는 카드가 있는데 AI 리포트엔 없어서 품질 역전 느낌.

## 💡 추가 아이디어

### 1. 상권 비교 기능
"이 위치 저장하고 다른 곳이랑 비교하기" — 2~3개 지점 나란히 비교하는 뷰. 창업자들의 핵심 니즈가 "어디가 더 나은가?"임.

### 2. 분석 결과 공유
카카오톡 / 링크 공유 버튼. "배우자한테 보내기", "동업자랑 같이 보기" 유즈케이스가 실제로 많음. 바이럴 효과도 있음.

### 3. 찜하기 / 히스토리
여러 상권 분석하다 보면 이전 분석이 기억 안 남. 최근 분석 목록 또는 북마크 기능이 있으면 재방문 유도 가능.

### 4. 업종별 상권 추천 역방향 플로우
"카페 창업하기 좋은 서울 상권 TOP 5 보여줘" — 지금은 위치 먼저 잡고 분석하는 구조인데, 반대로 업종 → 좋은 상권 추천 방향도 수요가 있음.

### 5. 잠긴 항목 블러 미리보기
잠금된 항목(생존율 분석, 리스크 경고 등)의 내용을 블러 처리해서 살짝 보여주면 궁금증 유발로 전환율 올라감. 지금은 자물쇠 아이콘만 있어서 안에 뭐가 있는지 모름.

### 6. "이 상권 위험해요" → 대안 상권 추천
분석 결과가 F등급일 때 "비슷한 조건에서 더 나은 상권 3곳" 추천. 유저 이탈 방지 + 재분석 유도.

### 7. AI 리포트에 인포그래픽 추가
종합 점수 게이지, 경쟁강도/상권활력 레이더 차트, 주요 수치 하이라이트 카드. PDF 공유 시 SNS 바이럴 가능성.
