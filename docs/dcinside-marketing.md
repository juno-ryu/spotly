## 글 선별 기준

### ✅ 댓글 달 것
- 최근 글 (당일~3일 이내)
- 창업 입지/상권 고민 언급
- 지역이나 업종 힌트가 있는 글

### ❌ 스킵할 것
- 2주 이상 오래된 글
- 업종/지역 힌트 전혀 없는 글
- 입지와 관련 없는 순수 자금/법인/투자 고민 글

---

## 댓글 작성 워크플로우

### 케이스 1: 지역 + 업종 모두 있음 (최우선)
1. 스팟리에서 해당 지역 + 업종으로 직접 분석 실행
2. 리포트 링크 + 실제 분석 결과 요약을 댓글에 포함

### 케이스 2: 지역 or 업종 하나만 있음
- 업종만 있으면 → 주거밀집 주요 상권으로 임의 지역 설정
- 지역만 있으면 → 해당 지역 메인 상권 + 프랜차이즈 인기 업종(치킨 등)으로 분석

### 케이스 3: 힌트 없음
- 기존에 생성된 리포트 링크 공유
- 스팟리 서비스 자체 소개

## 댓글 템플릿

### 기본형 (분석 후)
```
[지역] [업종] 입지분석 스팟리로 돌려봤는데
→ 종합 [등급] ([점수]/100점)
- 경쟁: [등급] (반경 300m 내 [N]개)
- 배후인구: [N]명
- [핵심 인사이트 1줄]

무료로 직접 해볼 수 있음: [리포트 링크]
스팟리(spotly.website)
```

### 서비스 소개형 (힌트 없을 때)
```
입지분석은 스팟리 써봐. 주소랑 업종 넣으면 AI가 경쟁업체 수, 배후인구, 유동인구, 리스크까지 다 분석해줌. 무료임
spotly.website
예시 리포트: [기존 리포트 링크]
```
