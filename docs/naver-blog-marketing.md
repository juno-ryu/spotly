## 발행 계획

- **빈도**: 주 2~3회
- **우선순위**: 유입 많았던 리포트부터 작성
- **키워드 선정**: 네이버 데이터랩 / 블랙키위로 검색량 확인 후 제목 결정

### 우선순위 선정 기준

1. **네이버 검색량 높은 지역** — SEO 유입 효과
2. **후킹 포인트가 강한 것** — 점수 극단값, 데이터 반전
3. **업종 다양성** — 카페만 쏠리지 않게
4. **서울 + 지방 균형** — 지방은 검색 경쟁 낮아 상위 노출 쉬움
5. **실제 유저가 분석한 지역** — 검색 수요 검증됨
