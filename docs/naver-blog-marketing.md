```

```

**예시:**
- 강남역 이자카야 창업 현실 | 월매출 6천만원인데 왜 폐업이 2배일까

- 건대입구 카페 창업 현실 | 월매출 2,712만원의 함정

**제목 원칙:**
- 핵심 키워드 "{지역} {업종} 창업" 반드시 포함
- 후킹 포인트: 숫자 + 반전/의문 형태
- 60자 이내 (검색결과에서 잘리지 않게)
- 연도 넣지 않기 (데이터 기준 시점 불명확)

### 본문 작성 원칙

1. **소제목마다 "{지역} {업종}" 반복** — 네이버 SEO 키워드 빈도 체크 대응
2. **숫자는 무조건 굵게** — 시선 끌기
3. **데이터 나열 금지** — 반드시 "그래서 뭐?" 해석을 붙일 것
4. **후킹 포인트 필수** — 함정, 반전, 소름, 충격 등 감정 유발
5. **리포트 실제 데이터만 사용** — 추측/과장 금지

## 발행 계획

- **빈도**: 주 2~3회
- **우선순위**: 유입 많았던 리포트부터 작성
- **키워드 선정**: 네이버 데이터랩 / 블랙키위로 검색량 확인 후 제목 결정

### 우선순위 선정 기준

1. **네이버 검색량 높은 지역** — SEO 유입 효과
2. **후킹 포인트가 강한 것** — 점수 극단값, 데이터 반전
3. **업종 다양성** — 카페만 쏠리지 않게
4. **서울 + 지방 균형** — 지방은 검색 경쟁 낮아 상위 노출 쉬움
5. **실제 유저가 분석한 지역** — 검색 수요 검증됨
