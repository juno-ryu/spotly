## 발견된 이슈

### ISS-01: 서울 지역인데 활력도/생존율 없이 인프라 지표만 표시 (건대입구역 술집)
- **케이스**: #5 (서울 광진구 아차산로 243 / 술집)
- **증상**: 서울 지역임에도 활력도, 생존율 지표가 빠지고 인프라 지표만 표시됨
- **기대**: 서울 골목상권 데이터가 있으므로 활력도, 생존율이 표시되어야 함
- **심각도**: HIGH - 서울인데 비서울 패턴으로 표시되면 분석 신뢰도 저하

### ISS-02: 잠실 베이커리 폐업률 0.0% 생존율 A등급 - 신뢰성 의심
- **케이스**: #4 (서울 송파구 올림픽로 지하 265 / 베이커리)
- **증상**: 폐업률 0.0%, 개업률 3.7%로 생존율 A등급
- **의심**: 폐업률 0.0%는 데이터 부재/수집 오류 가능성. 실제로 폐업이 0인 상권은 극히 드뭄
- **심각도**: MEDIUM - 데이터 자체는 골목상권 API 원본일 수 있으나, 0.0%인 경우 "데이터 부족" 경고가 필요

### ISS-03: 비수도권/비서울 지역 지하철 데이터가 주변 인프라 섹션에서 누락
- **케이스**: #8 (수원역), #9 (부평역), #10 (판교역)
- **증상**: 인프라 카드에는 "지하철 2역"으로 집계되지만, 하단 주변 데이터 섹션의 지하철 항목이 표시되지 않음
- **기대**: 지하철역 이름과 거리가 주변 데이터에도 표시되어야 함
- **심각도**: MEDIUM - 사용자가 어떤 역인지 확인할 수 없음

### ISS-04: 건대입구역 근처인데 건국대학교 표시 안됨

### ISS-06: 영등포역 편의점 생존율 데이터 누락

1. **서울 vs 비서울 지표 분기**: 서울은 활력도+생존율, 비서울은 인프라 지표를 사용하는 이중 구조가 명확히 동작하나, 건대입구역(서울)에서 인프라 패턴이 나온 것은 버그로 보임
2. ~~**소득 지표 미표시**~~: 소득 지표는 설계 단계에서 폐기됨 (KOSIS 데이터 부적합). 현재 스코어링에 미포함 정상.
3. **학교/대학교 데이터 희박**: 10건 중 학교 표시 3건, 대학교 표시 3건뿐. 반경 제한 문제일 수 있음
4. **비서울 지하철 표시 누락**: 비서울 지역에서 지하철 역 이름이 주변 데이터에 표시되지 않는 일관된 패턴

### ISS-15: 버스 노선 수 50개 상한 의심

- **심각도**: LOW - ISS-07(qa-issues.md)과 관련. 버스 노선 페이지네이션 미구현
