# 유튜브 쇼츠 대본 가이드

## 영상 구조

스팟리 실제 사용 과정을 화면 녹화한 영상. 나레이션 없이 텍스트 오버레이만 사용.
마지막 화면이 첫 화면과 동일하게 루프됨.

### 화면 순서 (고정)

1. 웰컴 인트로
2. 업종 선택
3. 지역 선택
4. 반경 선택
5. 분석 중 액션 (여기저기 찍어보는 장면, 길어서 텍스트 3개)
6. 로딩 (데이터 수집 중, 텍스트 3개)
7. 분석 결과 (상단 → 경쟁 → 매출 → 폐업률 순서로 스크롤)
8. 하단까지 스크롤 완료
9. 첫 화면으로 돌아가며 루프

## 텍스트 오버레이 작성 규칙

### 톤 & 스타일

- **대화체, 유머러스하게** — 딱딱한 설명 금지
- **ㄷㄷ, ㅋㅋ** 같은 표현 자연스럽게 사용
- 생각의 흐름을 그대로 나열하지 않음 (예: "여기로 찍어보자" ← 이런 당연한 말 금지)
- **궁금증 유발 → 답을 보여주는** 구조

### 구간별 규칙

**1. 웰컴 인트로 (후킹)**
- 시청자의 돈/리스크에 직접 와닿는 문구
- 예: "수천만원짜리 내 가게 차리기 전에 이거 꼭 보세요"
- ❌ "1억 넣고" (막연함) → ✅ "수천만원짜리 내 가게" (본인 일처럼)

**2~4. 업종/지역/반경 선택**
- 짧고 호기심 유발
- 예: "수유동 백반집 과연 될까?" / "내 가게 근처에 경쟁자 몇 명인지"
- ❌ "여기로 찍어보자" (당연한 말) / ❌ "서울 강북구 수유동 279-7" (주소 나열)

**5. 분석 중 액션 (길어서 3개)**
- 스팟리가 뭘 분석하는지 기능 소개
- 예: "경쟁업체 수, 유동인구, 매출, 폐업률" → "정부 공공데이터 6개 실시간 분석" → "AI가 전문가처럼 리포트까지 써줌"

**6. 로딩 (3개)**
- 실제 데이터소스를 구체적으로 언급 — 신뢰감 극대화
- 예: "국민연금 가입자 수로 실제 직원 수 파악" → "카카오맵에서 주변 경쟁점포 전수 조사" → "서울시 골목상권 카드매출 데이터 수집"

**7. 분석 결과 (핵심)**
- 수치 나열이 아니라 **감성적 판단 + 궁금증 유발**
- 결과 → 의문 → 답(이유) 순서로 전개
- 예:
  - 상단: "A등급 ㄷㄷ 근데 조건부래 뭔가 함정이 있다"
  - 경쟁: "한식집 7개에 프차 1개 싸울 만은 한데"
  - 매출: "월매출 3,553만 꽤 나오는데 왜 다들 나가떨어지지?"
  - 폐업률: "상권 축소 중 + 배후인구 2만명도 안 됨 결국 뜨내기 장사"
- ❌ "폐업률 4.2%" (숫자만) → ✅ "왜 다들 나가떨어지지?" (궁금증) → 이유 제시
- ❌ "돈 쓰는 층은 60대인데 지나가는 건 20대" (그래서 뭐?) → ✅ "배후인구 2만명도 안 됨 결국 뜨내기 장사" (신뢰감 있는 판단)

**8~9. 마무리**
- 행동 유도 (CTA) — 약간 도발적으로
- 예: "무료인데 안 써보고 수천만원 투자하는 건 좀..." → "감이 아니라 데이터로 판단하세요"

```
{후킹 문구} #{지역} #{업종} #창업 #상권분석 #소상공인 #자영업 #창업준비 #무료상권분석
```

### 설명란
```
{후킹 문구}

✅ 경쟁업체 수, 유동인구, 매출, 폐업률
✅ 정부 공공데이터 6개 실시간 분석
✅ AI가 전문가처럼 리포트까지

👉 무료 상권 분석
– 프로필링크 확인 –
– 고정댓글 확인 –

#창업 #상권분석 #{업종} #{지역} #소상공인 #자영업 #창업준비 #AI분석 #무료상권분석
```

### 소재 활용 구조

```
1. 후킹 (충격 데이터) — 0~3초
2. 스팟리에서 해당 지역 분석 시작 — 3~15초
3. 분석 중 (데이터소스 소개) — 15~25초
4. 결과 확인 + 감성적 판단 — 25~40초
5. CTA ("무료인데 안 써보고...") — 40~50초
```

### 우선순위

1번(가로수길), 2번(카페), 4번(배달앱)이 대중성 가장 높음.
5번(백종원)은 화제성 최고지만 저작권/명예훼손 리스크 있으니 팩트 기반으로만 언급.

## 새 영상 만들 때 체크리스트

1. 스팟리에서 분석 실행 (지역 + 업종)
2. 리포트 결과에서 핵심 수치 뽑기 (등급, 경쟁수, 매출, 폐업률, 특이사항)
3. 위 템플릿에 맞춰 텍스트 오버레이 작성
4. 화면 녹화 (스팟리 사용 과정)
5. 텍스트 오버레이 입히기
6. 고정댓글에 UTM 링크 등록
7. 업로드
