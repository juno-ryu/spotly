## ✅ curl 테스트 성공 (바로 연동 가능)

### 1. 소상공인 상가정보 API

| 항목 | 내용 |
|------|------|
| **data.go.kr ID** | 15012005 |
| **Base URL** | `http://apis.data.go.kr/B553077/api/open/sdsc2` |
| **인증** | `serviceKey` (`DATA_GO_KR_API_KEY`) |
| **호출 제한** | 30 TPS |
| **반경 조회** | ✅ `/storeListInRadius` (radius, cx, cy) |
| **curl 결과** | 강남역 500m: 전체 6,377건, 카페(I21201) 170건 |
| **현재 상태** | 코드 구현 완료 → 카카오와 중복으로 비활성 |

**엔드포인트 목록**:
- `/storeListInRadius` — 반경 내 점포 목록 (반경조회 ✅)
- `/storeZoneInRadius` — 반경 내 상권 영역 목록 (상권번호, 상권명, 폴리곤좌표 반환)
- `/storeZoneOne` — 상권번호로 상권 상세 조회
- `/largeUpjongList` — 업종 대분류 코드 목록

**필수 파라미터**: `cx`(경도), `cy`(위도), `radius`(m)
**선택 파라미터**: `indsLclsCd`(업종대분류), `indsSclsCd`(업종소분류), `pageNo`, `numOfRows`, `type`(json)
